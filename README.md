# github-learning-lab
# 📘 Learn GitHub – Beginner Playground

Welcome! This repo is designed for **absolute beginners** to learn Git & GitHub by doing.

## 🎯 What will you learn?
- Clone a repository
- Create a branch
- Make changes and commit them
- Open a Pull Request (PR)
- Merge a PR
- Push and pull changes

## 🛠️ Prerequisites
- A [GitHub account](https://github.com/join)
- Git installed on your computer ([download](https://git-scm.com/downloads))
