# 📘 Learn GitHub – Beginner Playground

Welcome! This repo helps beginners learn Git & GitHub.

## 🎯 What you'll learn
- Clone repositories
- Create branches
- Make commits
- Push to GitHub

## 📝 Exercise
1. Add your name file in `exercises/` folder
2. Commit your changes
3. Push to GitHub

## 📚 Git Commands
- `git status` - Check what changed
- `git add <file>` - Stage files
- `git commit -m "msg"` - Save changes
- `git push` - Upload to GitHub
